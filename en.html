Zator — A Language for AI Content Generation and Processing

A specialized programming language for building generative text and image pipelines via KoboldCpp and Stable Diffusion — simple, fast, and flexible.

context = "You are a game character designer."

var prompt = "Game character kobold-mage, clean green background"
var img = generate_img(prompt, context, 512, 512)

var cropped = chroma_key_crop(img, 10, 10, 3.0)

Features

Why Zator?

Zator is built for developers, designers, and content creators who need a simple, powerful, and reproducible way to work with AI.

?

Simple Syntax

Minimum boilerplate code — write generative scenarios as easily as regular scripts.

??

KoboldCpp Integration

Built-in functions for text and image generation via KoboldCpp and Stable Diffusion.

??

Image Processing

Color-based cropping (chroma key), scaling, and asset preparation directly within the language.

??

Control Structures

Conditions, loops, and user input allow for building interactive content generators.

??

File Handling

Save texts and images into your project structure without external libraries.

???

Pipeline Ready

Perfect for automating the generation of assets, stories, characters, and media.

Examples

Usage Examples

See how easy it is to create content with Zator — from stories to game assets.

Story Generation

context = "You are a fantasy writer."

var theme = "journey to the ancient dragon"
var prompt = "Write a short story about '{theme}'."
var story = generate_text(prompt, context, 250)

print(story)
save_txt(story, "stories/dragon_journey.txt")

Interactive Generator

context = "You are a content creation assistant."

print("Enter character description:")
input(desc)

var prompt = "Character portrait: {desc}, digital art"
var img = generate_img(prompt, context, 512, 512)

save_img(img, "characters/user_character.png")
print("Done!")

Batch Processing

context = ""

var img = generate_img("Sword icon on green background", context, 512, 512)
var cropped = chroma_key_crop(img, 5, 5, 4.0)

var icon64 = scale_to(cropped, 64, 64)
var icon128 = scale_to(cropped, 128, 128)

save_img(icon64, "icons/sword_64.png")
save_img(icon128, "icons/sword_128.png")
